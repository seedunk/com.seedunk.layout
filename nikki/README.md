# Nikki 
 ![bootstrap v4](../@web/img/bootstrap4.svg) ![bootstrap v4](../@web/img/jquery.svg) 

 Free Bootstrap 4 HTML5 Magazine Website Template，来自 [technext](https://github.com/technext/nikki) 

[预览](example/index.html) 

![Preview](Preview.png) 
 
## Key Features
 * Bootstrap 4
 * HTML5 & CSS3
 * Clean & minimal design
 * Fully responsive
 * Multi-page template
 * Split header
 * Slider on the header
 * Hover effects
 * CSS3 preloader
 * Megamenu
 * Zoom in effect
 * Call to action button
 * Drop-down menu
 * Breadcrumbs
 * Sticky top navigation bar
 * Search option
 * Blog section
 * Back to top button
 * Comment form UI
 * Contact form
 * Newsletter subscription form
 * Google Maps
 * 5+ UI elements
 * Fontawesome font icons

## Libraries and Plugins
